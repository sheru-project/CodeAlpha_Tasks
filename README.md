# 🛒 Simple E-commerce Store

A basic full-stack E-commerce web application that allows users to browse products, view detailed information, add items to a shopping cart, and place orders.

This project is designed to demonstrate fundamental concepts of frontend and backend development, including UI design, API integration, and state management.

---

## 🚀 Features

- 🛍️ Product Listing Page
- 🔍 Product Details View
- 🛒 Add to Cart Functionality
- ➕ Update / Remove Items from Cart
- 💳 Order Processing System
- 📱 Responsive Design

---

## 🛠️ Tech Stack

### Frontend:
- HTML5
- CSS3
- JavaScript (Vanilla JS)

### Backend:
- Node.js + Express.js  
  OR  
- Django (Python)project/
